# Advent of Code 2023 - Nim

Nim 2.0.0

## How to run
```bash
nimble run                    # Run all days
nimble run -- -d 2            # Run a specific day
nimble run -d:release -- -d 2 # Benchmark a specific day
nimble test                   # Run tests
```
